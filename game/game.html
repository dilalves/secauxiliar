Lady Bug em HTML

Esse protótipo foi refeito para ficar bem mais perto da vibe do original: proporção de tela arcade, labirinto com linhas finas, portas rotativas, bug box central, inimigos com cara retrô e movimentação estilo caça no corredor.

Não é emulação 1:1 da ROM. É uma recriação jogável em HTML5, inspirada nos gráficos e no layout do Lady Bug.

224 x 256 Portas giratórias Visual retrô HTML puro
Controles
Setas ou WASD para mover.
Espaço para reiniciar a fase.
Encoste na portinha amarela para ela girar e abrir outro caminho.
O que ficou mais próximo do original
Tela vertical arcade, trilhas finas, pontos no corredor, bônus piscando, caixa central dos insetos e a lógica de portas que muda o fluxo da perseguição.
